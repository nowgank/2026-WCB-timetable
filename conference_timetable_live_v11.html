마지막 업데이트: -
시트 컬럼(순서): 날짜 · 이름 · 세션시간 · 세션명 · 장소 · 발표시간 · 제목 · 비고
뱃지 숫자 = 해당 세션을 신청한 인원 수 · 초록 시간 = 개별 발표 시간
연동 방법 — ① 구글시트 1행에 날짜, 이름, 세션시간, 세션명, 장소, 발표시간, 제목, 비고 헤더를 이 순서대로 → ② 참가자가 관심 있는 발표마다 한 행씩 입력(같은 사람이 여러 행 가능, "Team SK"라는 이름으로 입력하면 해당 칸이 Team SK/참가자 두 영역으로 자동 분리되어 표시됨). 날짜는 2026-07-11처럼 실제 날짜로, 세션시간은 그 세션 전체 시간(예: 09:30-10:30), 발표시간은 그 발표 하나의 구체적인 시각(예: 09:50-10:00), 비고는 선택 입력으로 비워둬도 되며 입력하면 제목 옆 아이콘에 마우스를 올리거나 탭하면 내용이 보입니다 → ③ 파일 → 공유 → 웹에 게시 → 해당 시트만 CSV로 게시 → ④ 생성된 URL을 이 HTML 코드 상단의 const CSV_URL = "..." 부분에 붙여넣고 저장. Room 표시 순서는 코드 상단 const ROOM_ORDER = [...] 배열에 원하는 순서대로 적으면 항상 그 순서로 고정 표시되고(목록에 없는 Room은 뒤에 자동 추가), 각 Room은 세션 블록 왼쪽 강조선과 헤더 점 색으로 구분됩니다(최대 20개 Room까지 색이 자동 배정). 참가자 이름은 가나다순으로 색이 고정 배정되어 같은 사람은 항상 같은 색으로 보입니다. 전체 프로그램 링크는 코드 상단 const PROGRAM_URL = "..."에 학회 공식 페이지 주소를 넣으면 제목 옆 버튼이 활성화됩니다(비워두면 버튼이 자동으로 숨겨짐).
